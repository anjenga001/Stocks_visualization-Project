# Stocks_visualization-Project
## Overview
This project analyzes performance of software companies' stock performance from January 2021 to  December 2025. The dashboard proides the audience with insights into price trends, trading volume and volatility.

## Dashboard Preview

## Tech Stack
- Python (Data extraction and cleaning)
- PostgreSQL (table creation, data storage)
- Power BI (visualization, analysis)

## Data Pipeline
1. Data collected from Yahoo Finance API
2. Cleaned using Pandas
3. Created PostgreSQL database to normalize and store data as tables
4. Visualized in Power BI

## How to Use
1. Download '.pbix' file
2. Open in Power BI
3. Interact with featurea and visuals
